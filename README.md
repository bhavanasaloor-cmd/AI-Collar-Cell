
# AI Collar Cell

AI Collar Cell is a student healthcare innovation project that explores the use of wearable AI to assist doctors during consultations.

## Key Features
- AI-assisted speech-to-text documentation
- Draft digital prescription generation
- Secure patient record management
- Reduced administrative workload
- Doctor reviews and signs the prescription before it is handed to the patient

*This repository currently presents the project concept and documentation. Development is ongoing.
